# Overview

Wind Waker is a nodejs server, very simple to use yet powerful with optimized performance. It's easy to learn and used to build high quality Backend APIs. It's implemented with Typescript.

# Main concepts

  • Application
  • Actions
  • Pipes

You need to understand these 3 basics concepts, before start to use Wind Waker Framework.

# The Application

The Application is the core of a Wind Waker server, it will load all the configurations and Actions to create a Node.js server ready to receive incoming requests.

# The Actions

An Actions is a function defined by the user to handle an incoming request.

# The Pipes

A Pipe is a reusable function you can use to plugin logic. You can set Pipes at different levels: Application, Action Group or Action.
